== Description ==

<p>Plugin for adding latex code like [tex]\alpha[/tex] to a post. </p>


== Installation ==

1. Upload and extract `idienstlers-latex-code.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start to write a post and put your formula between [tex] and [/tex]
